เมนู Dashboard ใช้แสดงข้อมูลในรูปแบบของกราฟ

คำอธิบาย
  1. ช่วงวันที่ของข้อมูลที่ต้องการแสดง
  2. เลือกประเภทข้อมูลที่ต้องการแสดง ทั้งหมด หรือ รายบัญชี
    • ทั้งหมด หมายถึง แสดงจำนวนการบันทึกทั้งหมด เช่น 1 บัญชีอาจมีการบันทึกมากกว่า 1 ครั้ง
    • รายบัญชี หมายถึง แสดงจำนวนบัญชีที่มีการบัญทึก
  3. แสดงจำนวนการบันทึก
  4. แสดงชื่อผู้ใช้
  5. แสดงข้อมูลรวม
